# Session & Time-Based Behavior

## Core Principle
*A line drawn at 3AM means different from one drawn at 9AM.*

## Session Characteristics

| Session | Time (UTC) | Typical Behavior |
|---------|------------|------------------|
| **Asia (Tokyo)** | 00:00 - 09:00 | Ranging, low volume, accumulation |
| **London** | 07:00 - 16:00 | Trend setting, high volume |
| **New York** | 13:00 - 22:00 | Continuation or reversal, highest volume |
| **London-NY Overlap** | 13:00 - 16:00 | Maximum volatility |

## Session-Specific Memories

| Memory | Implication |
|--------|-------------|
| "Asia breakouts often fail" | Wait for London confirmation |
| "London sets the trend" | Trade with London direction |
| "NY often tests London high/low" | Expect retest in NY |
| "End-of-day moves fade" | Be cautious of 4PM+ moves |

## Time-Based Trap Patterns

| Pattern | Description | Agent Response |
|---------|-------------|----------------|
| **Asia Fakeout** | Break in Asia, reverse in London | "Wait for London confirmation" |
| **London Open Sweep** | London opens with stop hunt | "First 30 min often fake" |
| **NY Open Reversal** | NY reverses London trend | "Watch for reversal at 13:00 UTC" |
| **End-of-Day Trap** | 4PM move that fades | "Moves after 4PM often revert" |

## Time Memory: London Breakout → NY Retest

```
💾 **TIME MEMORY**

Historical observation:
- Breakouts established in London session are often retested in NY
- If retest holds, trend continues into NY close
- If retest fails, reversal likely

Current application:
- BTC broke resistance at $97,000 during London
- Current time: 14:00 UTC (NY just opened)
- Expect: Retest of $97,000 in next 1-2 hours

Recommendation: Wait for retest before adding to position
```

## Session-Based Drawing Strategy

### Asia Session (00:00-09:00 UTC)
- Mark levels but don't commit
- Any breakout is tentative
- Use for planning, not execution

### London Session (07:00-16:00 UTC)
- This is where trends are set
- Breakouts here are more reliable
- Volume confirms validity

### NY Session (13:00-22:00 UTC)
- Watch for London level retests
- Continuation or reversal zone
- End-of-day (20:00+) moves often fade

### Overlap (13:00-16:00 UTC)
- Highest volatility
- Breakouts can be explosive
- Also highest fake-out risk

## Time-Aware Workflow

Before validating any drawing:
1. Check current session
2. Check when drawing was created
3. Apply session-appropriate confidence
4. Wait for confirmation if in weak session

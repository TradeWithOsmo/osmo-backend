# Market Regime Memory

## Core Principle
*Markets rhyme, not repeat exactly.*

## Common Regime Patterns

| Regime | Typical Behavior | Drawing Implications |
|--------|------------------|----------------------|
| **Strong Trend, Shallow Pullbacks** | 20-30% retracement max | Fib 0.382-0.5 entries, not 0.618 |
| **Healthy Trend, Deep Pullbacks** | 50-61.8% retracements | Standard Fib levels valid |
| **Wide Range, High Volatility** | Large swings within bounds | Wider zones, not tight lines |
| **Tight Compression** | Narrowing, coiling | Triangle, expect expansion |
| **News-Driven Chaos** | Unpredictable spikes | Avoid drawing, wait for settle |
| **Liquidity Hunt Mode** | Fake breaks, stop runs | Don't trust first break |

## Pattern Memory: Strong Trend + Shallow Pullbacks

```
💾 **REGIME MEMORY**

Historical observation:
- In strong uptrends, price often only pulls back to 0.382-0.5 Fib
- Waiting for 0.618 often means missing the move
- Trendlines drawn from shallow lows get many touches

Implication for current setup:
- Current trend is strong (ADX > 30, clean HH/HL)
- Consider 0.5 Fib entry instead of waiting for 0.618
- Trendline from recent shallow pullback is likely valid
```

## Session-Based Regime Memory

### Session Patterns

| Observation | Memory |
|-------------|--------|
| "Trendline breaks in Asia often reverse in London" | Don't trust Asia breaks |
| "London sets the trend, NY tests it" | Trade with London direction |
| "End-of-day moves often fade next day" | Be cautious of 4PM moves |

### Agent Application

Before validating a break: "Wait, this happened in Asia session. Historically, these tend to reverse in London."

## Volatility Regime Memory

### Volatility Patterns

| Condition | Typical Outcome |
|-----------|-----------------|
| ATR compression (3+ days) | Expect expansion soon |
| ATR expansion after news | Usually normalizes in 24h |
| Low volume + tight range | Breakout coming |

## Example Response

**Q:** "Pola ini sering muncul di kondisi apa?"

**A:** "This looks like a compression regime. In my memory, when ATR drops below 0.5% for 3+ days with a narrowing triangle, a significant move usually follows within 24-48 hours.

The direction is often random, so I recommend:
1. Wait for breakout confirmation
2. Set alerts at triangle boundaries
3. Don't trade inside the compression"

## Application Checklist

When analyzing current market:
1. Check ATR history (compressed or expanded?)
2. Identify pullback pattern (shallow or deep?)
3. Note session timing
4. Compare to regime library
5. Adjust drawing approach accordingly

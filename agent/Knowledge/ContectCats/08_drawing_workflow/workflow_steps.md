# Drawing Decision Workflow

## Core Principle
*Think before you draw.*

## 5-Step Pre-Drawing Workflow

### STEP 1: CONTEXT ✅
- What is the market condition?
- What timeframe am I on?

### STEP 2: PURPOSE ✅
- What am I trying to mark? (Trend? Level? Zone?)
- What decision will this drawing help with?

### STEP 3: ANCHOR POINT ✅
- Where should this drawing start?
- Is this a significant point (swing high/low)?

### STEP 4: DIRECTION & EXTENSION ✅
- How should this drawing extend?
- Finite line or infinite ray?

### STEP 5: VALIDATION ✅
- Does this line respect price action?
- How many touches confirm it?

## Anchor Point Selection

| Drawing | Anchor Point |
|---------|--------------|
| `trend_line` | Swing lows (uptrend) or swing highs (downtrend) |
| `horizontal_line` | Clear reaction point (bounce/rejection) |
| `fib_retracement` | Impulse start (low for up, high for down) |
| `rectangle` | Range high and range low |

## Validation Criteria

### Touch Count Validity

| Touches | Validity |
|---------|----------|
| 1 | Not valid (just a point) |
| 2 | Minimum valid (hypothesis) |
| 3+ | Confirmed (strong) |

### Validation Checks

Ask these questions:
- "Does price respect this line?"
- "Are there clean reactions (bounce/rejection) at this level?"
- "Is this line diagonal when market is ranging?" (Invalid)
- "Is this line horizontal when market is trending?" (May be key level)

## Post-Drawing Evaluation

### Line Invalidation Conditions

| Condition | Status |
|-----------|--------|
| Price closes beyond line | Potentially invalid |
| Multiple closes beyond | Invalid (remove or update) |
| Wick spike then reversal | Still valid (liquidity grab) |
| Break + retest holds | New role (S/R flip) |

### Re-evaluation Triggers

When to check if drawing is still valid:
- New swing point formed
- Price breaks the line
- Timeframe change
- Significant time passed

## Example Validation

**Q:** "Ini garis masih valid atau udah rusak?"

**A:** "Let me check the validation criteria...

The line was drawn connecting 2 swing lows. Since then:
1. Price touched it 2 more times (now 4 touches ✅)
2. Most recent touch was a clean bounce ✅
3. No candle has closed below it ✅

**Verdict: Line is still valid and getting stronger with each touch.**"
